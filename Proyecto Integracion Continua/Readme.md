# Convertidores de Unidades

Este proyecto contiene cuatro convertidores de unidades de medida implementados en C# utilizando Windows Forms.

## Conversor de Peso

El **Conversor de Peso** permite convertir entre kilogramos (kg) y libras (lb). Simplemente ingresa un valor en una unidad y obtendrás la conversión inmediata en la otra unidad.

## Conversor de Longitud

El **Conversor de Longitud** te permite convertir entre kilómetros (km) y millas (mi). Ingresa un valor en una unidad y el convertidor te mostrará la conversión en la otra unidad de medida.

## Conversor de Velocidad

El **Conversor de Velocidad** es capaz de convertir entre tres unidades de velocidad: kilómetros por hora (km/h), millas por hora (mph) y metros por segundo (m/s). Ingresa un valor en una unidad y obtén la conversión instantánea en las otras dos unidades.

## Conversor de Tiempo

El **Conversor de Tiempo** permite convertir entre horas, minutos, segundos y días. Ingresa un valor en una unidad de tiempo y obtén la conversión automática en las otras unidades de tiempo.


## Cómo utilizar los convertidores

1. Abre la aplicación y selecciona el convertidor que desees utilizar en la barra de navegación.

2. Ingresa un valor en el campo de entrada correspondiente a la unidad de medida que desees convertir.

3. Haz clic en el botón "Convertir" para obtener la conversión.

4. Si deseas borrar los campos de entrada y resultados, puedes hacerlo con el botón "Limpiar".

## Requisitos

- Windows
- .NET Framework (la versión de .NET Framework dependerá de la configuración del proyecto)

## Configuración del proyecto

Este proyecto está construido con Windows Forms y Visual Studio. Puedes abrirlo en Visual Studio y ejecutar la aplicación desde allí.
